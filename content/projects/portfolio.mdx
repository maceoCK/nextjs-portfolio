This personal portfolio website is a showcase of modern web development techniques and technologies. Built with Next.js, React, and Tailwind CSS, it offers a fast, responsive, and visually appealing platform to display projects and skills.

## Key Features and Technologies

- **Next.js**: Utilizing the power of server-side rendering and static site generation for optimal performance and SEO.
- **React**: Building interactive and reusable UI components for a smooth user experience.
- **Tailwind CSS**: Implementing a utility-first CSS framework for rapid and consistent styling.
- **TypeScript**: Ensuring type safety and improving code quality throughout the project.
- **Contentlayer**: Managing and rendering content with ease, allowing for efficient updates to project information.
- **Framer Motion**: Adding subtle animations to enhance the user interface and engagement.
- **Vercel Hosting**: Leveraging Vercel's platform for seamless deployment and hosting, ensuring high availability and performance.

## Performance and Analytics

- **Vercel Analytics**: Integrated for tracking page views and user interactions.
- **Vercel Speed Insights**: Monitoring and optimizing website performance.

## Content Management

- **MDX**: Using MDX for writing content, combining the power of Markdown with the flexibility of JSX.
- **Dynamic Project Pages**: Automatically generating pages for each project using Next.js dynamic routing.
